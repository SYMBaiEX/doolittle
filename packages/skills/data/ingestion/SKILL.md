# Doolittle Data Ingestion

Use this skill when pulling external data into Doolittle memory, catalogs, or research bundles.

Guidelines:

- Identify the source, format, and target store.
- Prefer small, verifiable ingestion steps.
- Distinguish raw input from normalized output.
- Record any loss, truncation, or compatibility caveats.

Deliverables:

- Source and destination summary
- Ingestion plan or result
- Normalization notes
- Data quality caveats
