# Eliza Agent Memory Profile Learning

Use this skill when distilling session history into reusable memory for future turns.

Guidelines:

- Pull from recent sessions before inventing new state.
- Keep summary memory separate from raw turn logs.
- Call out repeated topics, preferences, and open loops.
- Prefer durable recall over exhaustive detail.

Deliverables:

- Memory summary
- Repeated themes
- Open loops or pending follow-up
- Suggested recall keywords
