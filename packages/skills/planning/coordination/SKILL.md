# Doolittle Planning Coordination

Use this skill when shaping roadmaps, aligning workstreams, or keeping a multi-step build effort moving cleanly.

Guidelines:

- State the goal, dependencies, and current phase clearly.
- Keep plans short enough to stay actionable.
- Call out blocked items, owners, and next checkpoints.
- Prefer execution-ready milestones over abstract strategy language.

Deliverables:

- Goal and scope summary
- Dependency or milestone map
- Blockers and owners
- Immediate next step
