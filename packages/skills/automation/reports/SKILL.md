# Automation Reports

Use this skill when creating scheduled summaries, cron jobs, or recurring operational reports.

Guidelines:

- Define the cadence clearly.
- State the delivery target explicitly.
- Keep recurring report outputs compact and structured.
- Prefer durable, repeatable prompts over one-off conversational phrasing.

Deliverables:

- Schedule description
- Report prompt
- Delivery target
- Success criteria for each run
