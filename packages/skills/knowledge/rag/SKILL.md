# Doolittle Knowledge Retrieval

Use this skill when ingesting documents, retrieving supporting context, or wiring Doolittle memory to a source-backed answer.

Guidelines:

- Distinguish source material from interpretation.
- Prefer retrieval-backed notes over speculative summaries.
- Call out confidence, recency, and citation quality.
- Keep the working set small and explain what was searched.

Deliverables:

- Source list or corpus scope
- Retrieved context summary
- Confidence or coverage notes
- Follow-up gaps to ingest or verify
