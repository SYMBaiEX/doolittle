# Doolittle Batch Research

Use this skill when reviewing multiple trajectories, replays, or evaluation bundles at once.

Guidelines:

- Define the batch scope before analysis starts.
- Group findings by pattern rather than by individual run when possible.
- Highlight regressions, repeat wins, and reusable fixes.
- Keep the final summary operational and short.

Deliverables:

- Batch scope
- Review artifacts
- Repeat patterns or regressions
- Next action
