# Eliza Agent Action Bench Research

Use this skill when you need to evaluate action-calling breadth, benchmark coverage, or the impact of benchmark plugins on the runtime surface.

Guidelines:

- State which benchmark family is being exercised.
- Compare coverage before and after any plugin or runtime change.
- Prefer concrete action counts, manifests, and replay artifacts.
- Call out when benchmark actions should stay disabled outside evaluation.

Deliverables:

- Benchmark family and scope
- Expected action coverage
- Evaluation or replay artifacts
- Regression notes or rollout guidance
