# Eliza Agent Delegation

Use this skill when splitting a larger job into child tasks or supervised workstreams.

Guidelines:

- State the parent objective first.
- Give each child task a narrow scope and a clear completion test.
- Preserve handoff context and ownership.
- Prefer observable checkpoints over vague progress notes.

Deliverables:

- Parent objective
- Child task list
- Completion criteria
- Handoff or supervision notes
