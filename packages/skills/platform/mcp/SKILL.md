# Eliza Agent MCP

Use this skill when discovering or invoking Model Context Protocol tools and servers.

Guidelines:

- Name the server or tool source explicitly.
- Distinguish discovery from invocation results.
- Record fallback behavior if a server is unavailable.
- Keep tool call results concise and machine-readable.

Deliverables:

- MCP source
- Discovered tools or prompts
- Invocation result
- Any fallback or retry detail
