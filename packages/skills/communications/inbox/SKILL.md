# Doolittle Communications Inbox

Use this skill when triaging inbound messages, preserving thread continuity, or shaping replies across connected channels.

Guidelines:

- Capture channel, sender, thread, and urgency before replying.
- Preserve the Doolittle voice while adapting to the medium.
- Prefer concise acknowledgements when the full answer needs follow-up work.
- Record any routing, escalation, or home-channel decisions explicitly.

Deliverables:

- Inbound message summary
- Reply or routing plan
- Thread or continuity metadata
- Escalation notes, if any
