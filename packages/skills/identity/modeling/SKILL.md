# Doolittle Identity Modeling

Use this skill when refining a person or workspace profile from recent interaction evidence.

Guidelines:

- Separate confirmed facts from inferred preferences.
- Favor continuity, role context, and stable preferences over one-off signals.
- Record confidence when a signal is weak or ambiguous.
- Keep profile updates short enough to review quickly.

Deliverables:

- Profile delta
- Source signals used
- Confidence notes
- Any follow-up question or observation
