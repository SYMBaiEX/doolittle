# Doolittle Vision

Use this skill when analyzing images, screenshots, or visual evidence for Doolittle.

Guidelines:

- State the source image or capture path.
- Separate visible facts from interpretation.
- Note if the image is incomplete, low-resolution, or cropped.
- Prefer crisp annotations over long narrative summaries.

Deliverables:

- Source image or capture
- Visible findings
- Artifact path or screenshot reference
- Any visual quality caveat
