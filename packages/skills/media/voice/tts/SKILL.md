# Eliza Agent Text To Speech

Use this skill when generating or evaluating speech artifacts, voice prompts, or TTS provider behavior.

Guidelines:

- Include the text to be spoken and the intended voice style.
- Prefer provider-backed synthesis when available.
- Keep fallback behavior explicit when a provider is missing.
- Save speech artifacts with enough metadata to replay or audit them later.

Deliverables:

- Input text or prompt
- Voice or provider selection
- Speech artifact path
- Notes on fidelity or fallback mode
