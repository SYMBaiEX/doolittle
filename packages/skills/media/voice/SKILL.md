# Eliza Agent Media and Voice

Use this skill when handling audio, video, images, speech synthesis, or transcription work for Eliza Agent.

Guidelines:

- State the media type and the intended output clearly.
- Prefer transcript, caption, and artifact paths over vague summaries.
- Separate human-facing synthesis from machine-readable extraction.
- Mention quality limits when audio or image fidelity is imperfect.

Deliverables:

- Source media
- Transcript, caption, or analysis output
- Artifact location
- Any quality or fidelity caveat
