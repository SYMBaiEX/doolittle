# Doolittle Browser Research

Use this skill when capturing web pages, comparing browser artifacts, or collecting evidence from a remote page.

Guidelines:

- Capture the URL, timestamp, and artifact type explicitly.
- Prefer reproducible evidence over prose-only summaries.
- Distinguish snapshot, screenshot, and extracted text output.
- Note browser backend limitations when they affect the result.

Deliverables:

- Target URL or page set
- Capture artifact paths
- Key observations or deltas
- Any browser limitations or follow-up checks
