# Doolittle Regression Testing

Use this skill when protecting Doolittle behavior with repeatable tests, smoke checks, or regression comparisons.

Guidelines:

- Capture what changed and what should stay stable.
- Prefer narrow, reproducible checks over broad speculation.
- Call out fixtures, mocks, and runtime assumptions.
- Tie the result back to the exact user-visible surface.

Deliverables:

- Regression scope
- Test or smoke plan
- Expected behavior
- Follow-up coverage gaps
