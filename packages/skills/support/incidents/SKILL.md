# Doolittle Incident Support

Use this skill when triaging outages, degraded behavior, or operator-visible regressions that need a fast, calm response.

Guidelines:

- Capture symptom, scope, and start time first.
- Separate user impact from root cause speculation.
- Prefer containment and restoration before deep diagnosis.
- Record follow-up items for the post-incident review.

Deliverables:

- Incident summary
- Impacted surface area
- Containment or mitigation action
- Follow-up investigation list
