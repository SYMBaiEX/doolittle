---
command: /review
title: Repo Review
description: Review the current repo or target codebase and return the important structure, entrypoints, and risks.
---
Review {{TARGET}} in the local workspace.

Inspect first, then answer directly.

Return:
- a short overview of what the repo is for
- the top-level structure that matters
- the key entrypoints, scripts, and packages
- the biggest risks, rough edges, or surprises
- where a new contributor should start

Keep it concise, grounded, and terminal-friendly.
